cidr-converter

With CIDR, you can write compact IP addresses along with their subnet masks. Using this converter, you can pack and unpack IP addresses and their subnets to the corresponding CIDR notation, and vice-versa. We currently support IPv4 only, but support for IPv6 is in the works.

Learn more about CIDR »

Start with typing an IP address and subnet, or paste a CIDR notation on the right. The other fields will populate automatically.

How do we do this?

The conversion is done with JavaScript.
We published cidr-converter at Github, licensed under GPL. This means you can use it for free in your own open-source projects.
